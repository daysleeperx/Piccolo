Piccolo is an algorithmic music generator that supports integration with Sonic Pi, utilizing the Open Sound Control protocol. Melody variations can be generated from MIDI sources and from messages sent by Sonic Pi directly in real-time. Algorithmic music composition implementations are based on Markov chains and [Google AIâ€™s Magenta](https://magenta.tensorflow.org/) neural network.
